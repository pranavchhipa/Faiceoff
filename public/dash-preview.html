● Shared rules across roles

Same palette · different rhythm.

  • Admin: three-pane split, list-detail power tool — feels like Figma/Superhuman.
  • Mobile: bottom 5-tab bar + gold FAB. Bottom-sheet drawers for details.
  • Studio Black palette unchanged. Only Chrome + density + rhythm differ per role.
